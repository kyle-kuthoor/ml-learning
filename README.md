# Machine Learning Basics — Linear Regression

## Overview
An introduction to supervised machine learning using Scikit-learn, 
applied to a real social media advertising dataset (300,000 campaigns).

## What it covers
- Feature selection and target variable definition
- Train/test split (80/20)
- Linear Regression model training and evaluation
- R² score and Mean Squared Error interpretation
- One-hot encoding of categorical variables
- Feature coefficient analysis

## Key findings
- Model without platform features: R² = 0.2462
- Model with platform features: R² = 0.3325 (35% improvement)
- Pinterest coefficient of -3.28 confirms it as the dominant 
  negative ROI driver across all campaigns

## Tools
Python, Pandas, Scikit-learn, Matplotlib, Seaborn

## Dataset
Social Media Advertising Dataset — Kaggle (300,000 rows)
